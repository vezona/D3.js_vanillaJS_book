d3 Time Formats

1. d3.timeParse( ) ⇒ 將日期等資訊轉換成 D3 看得懂的數值

程式碼

            
              //JS
              const timeData = "2023-03-07";
              const timeParse = d3.timeParse("%Y-%m-%d");
              console.log("timeParse", timeParse(timeData)); 
              // Tue Mar 07 2023 00:00:00 GMT+0800 (台北標準時間)
