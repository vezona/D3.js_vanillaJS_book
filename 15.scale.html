比例尺 Scales

輸入域與輸出域 Domain ＆ Range
  • 輸入域 Domain 是在進行比例尺換算前，資料的整個數值範圍
  • 輸出域 Range 則是進行比例尺換算後，得到換算之後的資料數值範圍 
            
              // 輸入與輸出比例換算範例
              const convert = d3.scaleLinear().domain([0, 100]).range([0, 10]);
  
              console.log(convert(50)); // 5，換算輸出比例完成
連續性與離散性 continuous & discrete
  • 連續性 continuous 指的是資料之間具備關聯的特性，可以用某些運算方式找出彼此的關聯，這類資料通常為數字、日期等等
  • 離散性 discrete 指的是則是資料之間並沒有任何關聯，無法用任何運算方式找出彼此的關聯，這類資料通常為字串

將連續性或離散性搭配輸入域與輸出域的概念，就可以得出四種結果

  • continuous input 輸入的資料是連續性資料
  • continuous output 輸出的資料是連續性資料
  • discrete input 輸入的資料是離散性資料
  • discrete output 輸出的資料是離散性資料
